什么是Colab？要谈论有关colab的好处，我们需要谈论编程编辑器。 通常，当我们编程时，我们编写一个程序并执行它。 我们需要应对的挑战：如果我们要执行部分程序以查看结果是什么， 该怎么办？因此，可以通过像Colab这样的笔记本来解决这一挑战。 Colab允许用户执行部分代码，并允许他们编写markdown嵌入式文本编辑器， 以供用户用来对程序进行描述。即，Colab 笔记本使测试代码非常容易。 Colab shortcuts:
  • Ctrl m m: to change a cell to a text cell
  • Ctrl m y: to change a cell to a code cell
  • Ctrl m d: to delete a selected cell
  • Ctrl m a: to insert a code cell above
  • Ctrl m b: to insert a code cell below
  • Alt Enter: to run a cell and create a new cell
  • Ctrl enter: run a cell and stay
