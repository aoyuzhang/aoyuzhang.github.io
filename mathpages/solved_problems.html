Prove that given \(n\) cities, where between each pair of city, there is a unidirectional road, there must be a city where it is possible to travel there from any other city by passing by at most one other city.
We can model this problem using a directed graph, Let's see an example:
We consider the city with the maximum number of incomming roads. In the above graph representation of roads between cities, the number next to the city name is the number of incomming roads and we see city B has the most number of incomming roads. So then if you start at any other city, you get to city B using two roads.
The reason is that if there is a city D that does not have a directed road to city B, then there a city say C that goes to B and D goes to C. If all roads to goes to B goes to D, then D has more incomming roads than B, contradiction.
