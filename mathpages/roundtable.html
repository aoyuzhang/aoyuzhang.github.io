A group of \(n\) people sits down, uniformyly at random around a round table. What is the probability that two person A,B sits next to each other?
The solution is just that say we want to know the probability that person A and person B sits next to each other, then out of the \(n-1\) places, only two are next to person A, So the probability that person B sits next to person A, is just the probability of choosing the two places that are adjacent to person A out of the \(n-1\) choices and that gives: \(\frac{2}{n-1}\).
Now consider \(ab\) couples sitting at \(a\) tables with \(2b\) seats each. What is the probability that there are in total exactly \(k\) couples sitting at the same talbe?
\(\textbf{Solution:}\)
A couple is called good if they sit on the same table. Let \(C\) be the event that the couples \(1,...,k\) sits on the same table and the rest of couples don't sits on the same table. Ok so now, the probability that there are exactly \(k\) good couples is going to be \(\binom{ab}{k}P(C)\). Ok so we just need to compute \(P(C)\). Let's denote \(\pi_n\) the probability that couples \(1,...,n\) are good couples. Ok now we define the following events:
  • \(A = \{ \text{couples }1,...,k\text{ are good} \} \)
  • \(A_n = \{ \text{couples }1,...,k,\text{and n are good} \} \)
So notice that the events \(A_i\) are subsets of event \(A\). So now, the event \(C\) is just event \( A- \cup_{i=k+1}^{ab} A_i \). And so we have \[P(C)=P(A) - P(\cup_{i=k+1}^{ab} A_i)\] So then we use the inlusion exclusion principal to find \(P(C) = P(A) - \sum_{t=1}^{ab - k} \sum_{i_1,...,i_t}(-1)^{t+1}P(A_{i_1} \cap ... \cap A_{i_t}) \) Ok so \(P(A_{i1} \cap ... \cap A_{it}) = \pi_{k+t} \) by definition. There are \(\binom{ab-k}{t}\) terms of the form in the summation. Sow we get: \[P(C) = \sum_{t=0}^{ab-k}(-1)^t \binom{ab-k}{t} \pi_{k+t} \]. So now, we just have to calculate \(\pi_n\).
Ok so we have \(2ab\) people and we want the first \(n\) couples to be sitting on same table. Ok let's denote \(n_1,n_2,...n_a\) to be the number of couples that sits on table \(1,...,a\). Ok There are \(\binom{n}{n_1,...,n_a} \) ways to put the \(n\) couples around the \(a\) tables. Then there are \(\binom{2ab-2n}{ab-2n_1,...,2b-2n_a} \) ways to choose table for the remaining people to sit on the table. So the number of ways that the first \(n\) couples are good is \[ \sum_{n_1+...n_a=n} \binom{n}{n_1,...,n_a} \binom{2ab - 2n}{2b-2n_1,...,2b-2n_a} \] Ok so the total number of ways to assign table to the people is \(\binom{(2ab)!^a}{(2ab)!}\)
