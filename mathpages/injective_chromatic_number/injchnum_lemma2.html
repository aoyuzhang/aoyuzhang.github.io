\(\textbf{3) }\) If \(G\) is a graph of diameter \(2\) whose independence number is \( \alpha(G)=p\), then \(p \leq \chi_i(G)\).
Let's start with a very simple example:
Recall that the denifinition of an independent set of a graph \(G\) is:
A subset of the vertex set of \(S \subset V(G)\), where \(\forall u,v \in S, (u,v) \notin E(G)\).
The size of a largest independent set of a graph is denoted by \(\alpha(G)\)
In general, a graph could have many maximum independent set, but here, the maximum independent set \(S \subset V(G) \) is: \[S=\{E,A,C\}\] Notice that any two vertices in \(S\) is neighbhor to some vertex in \(V(G)-S\): \[\begin{cases} E A, \text{ neightbhors of } D\\ E C, \text{ neightbhors of } D\\ C A, \text{ neightbhors of } B \end{cases} \] This is because the diameter of the graph is \(2\) and so any two no adjacent verteces must be connected via a path of length \(2\). Therefore, they must have a common neighbhors.
Now since any vertices in the independent set \(S\) have a common neighbhor, they need to be colored differently in an injective coloring. Let's consider vertices \(E,A\):
Now, when consider the coloring of the vertex \(C\), well since \(C\) and \(A\) needs different coloring and \(C\) and \(E\) needs different coloring, we must give \(C\) a third color:
And we need \(|S|\) colors to color the vertices in the largest independent set of the graph alone. Therefore, the injective chromatic number of the graph should be at least as large as the size of the laegest independent set of \(G\): \[\chi_i(G) \geq \alpha(G)\] To give a coloring procedure, given an arbitary simple graph \(G\), let \(S \subset V(G)\) be a largest independent set of \(G\), namely \(\alpha(G)=|S|\). Then Enumerate vertices in \(S\): \[S=\{v_1,v_2,...,v_{\alpha(G)} \} \] Then color \(v_1\) arbitrary,
color of \(v_2\) needs to differ from that of \(v_1\),
color of \(v_3\) needs to differ from that of \(v_1,v_2\),
and so on until:
color of \(v_{\alpha(G)}\) need to differ from that of \(v_1,v_2,...,v_{\alpha(G)-1}\)
And so we needed at least \( \alpha(G)\) colors for an injective coloring of \(G\).
