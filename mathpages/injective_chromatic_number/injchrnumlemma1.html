\(\textbf{1) }\)Given \(d\)-regular graph \(G\) with \(\chi_i(G)=d\), then \(d\) is a divisor of size of vertex set of \(G\), \(d||V(G)|\)
Let's consider as an example, a \(3\)-regular graph with a \(3\)-injective coloring:
Let's denote the set of vertices with color, say pink, by \(V_{pink}\): \[V_{pink}=\{A,F\} \] For each vertex, we count its number of pink neighbhors: \[ \begin{cases} A: \text{ connected to pink vertex } F, 1 \text{ pink neighbhor}\\ B:\text{ connected to pink vertex } A, 1 \text{ pink neighbhor}\\ C:\text{ connected to pink vertex } F, 1 \text{ pink neighbhor}\\ D:\text{ connected to pink vertex } A, 1 \text{ pink neighbhor}\\ E:\text{ connected to pink vertex } F, 1 \text{ pink neighbhor}\\ F:\text{ connected to pink vertex } A, 1 \text{ pink neighbhor} \end{cases} \] Namely, each vertex has exactly one neighbhor of pink color and each pink vertex are neighbhors to \(3\) vertices. \[\begin{cases} F: \{A,C,E \}\\ A: \{B,D,F \} \end{cases} \] The reason is because of injective coloring and the fact that the degree of each vertex is \(3\).
So the following is true: (number of different colors) \(\cdot\) (number of pink vertices) \(=\) (total number of vertices): \[3 \cdot 2 =6\] And so here number of colors, which is \(3\), is a divisor of total number of vertices, which is \(6\).
We create the proof for arbitary \(d\) injective coloring of \(d\) regular graph \(G\) by miniking the above example.
So given an \(d\)- injective coloring of \(G\), we think of each colors as a number in \(\{1,...,d\}\).
We denote the set vertices with color \(i\) by \(V_i\).
Then, by the same reasonning as above example, we get: \[ |V(G)|=|V_i| \cdot d \] Hence this shows that \(d\) is a divisor of \(|V(G)|\).
