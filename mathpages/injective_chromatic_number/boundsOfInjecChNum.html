For a cycle of length \(n\), we have: \[\begin{cases} \chi_i(C_n)=2, n \equiv 0(\mod 4)\\ \chi_i(C_n)=3, else. \end{cases} \]
This is because the common neighbhor graph of a cycle of length \(n\) is: two cycles of length \(\frac{n}{2}\) if \(n\) is even or a cycle of length \(n\) if n is odd. We know that the injective chromatic number of a graph is the chromatic number of its common neighbhor graph, And since the common neighbhor graph consists of cycle of even length if and only if the graph is a cycle of lenght, a multiple of \(4\). And the common neighbhor graph of a cycle of odd length is a cycle of odd length. We have that claim using the fact: \[ \begin{cases} \chi(C_n)=2, n \equiv 0 (\mod 2)\\ \chi(C_n)=3, n \equiv 1 *\mod 2) \end{cases} \]
