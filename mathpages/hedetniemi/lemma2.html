(lemma 2) Hedetniemi's conjecture is equivalent to : \[ \chi(G)>n \Rightarrow \chi((K_n)^G) \leq n\]
So given that \(\chi(G)>n\), \[\begin{gather*} \chi((K_n)^G) \leq n, \\ \iff (K_n)^G \to K_n \tag{Proper n coloring is equivalent to a homomorphism to \(K_n\)}\\ \iff \forall H, (H \to (K_n)^G \Rightarrow H \to K_n) \tag{ if direction is because Homomorphism is a transitive relation, only if direction is by taking \(H=(K_n)^G\)} \\ \iff \forall H, (G \times H \to K_n \Rightarrow H \to K_n) \tag{Applying previous lemma}\\ \iff \forall H, (\chi(G \times H) \leq n \Rightarrow \chi(H) \leq n) \tag{Proper n coloring is equivalent to a homomorphism to \(K_n\)}\\ \iff \text{Hedetniemi's conjecture is true.} \tag{ Since \(\chi(G)>n\)} \end{gather*}\]
