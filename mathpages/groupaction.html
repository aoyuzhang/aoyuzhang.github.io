How many different necklaces, up to rotation, you can make with the gems. In this paper, we'll answer this question by using a powerful technique of group action and see its application in a more general setting. Fist of all,let's see an example of making a new set out of two old ones:
Given two set \(G,X\), their \(\textbf{Cartesian product}\) is denoted by and defined as: \[G \times X := \{(g,x)| g \in G, x \in X\}\] Namely, the cartesian product of two set is a set of ordered pair, where the first element belongs to \(G\) and the second element belongs to \(X\).
For example, if we have \(G=\{1,2,3\}, X= \{a,b\}\) then \( G \times X =\{(1,a),(1,b),(2,a),(2,b),(3,a),(3,b)\} \).
Next, we need to be familia on how we define a function, when we write: \[f:A \to B\] It means \(f\) is a function whose domain is \(A\) and whose codomain is \(B\).
OK, to tell you about how group action solves our problem, we first need to define it:
(Definition 2.3) Given a group \(G\) and a set \(X\), a \(\textbf{group action}\) is a function \(\phi : G \times X \to X\) such that:
  1. \(\forall x \in X, \phi((e,x)) = x\). Here \(e\) is the identity element in \(G\)
  2. \(\forall g,h \in G, x \in X, \phi(gh,x)= \phi(g,\phi(h,x)) \)
For simplicity, we denote \(\phi(g,x)\) as simply \(gx\). we also denote action of \(G\) to \(X\) by \(G \curvearrowright X\).
Alright, the big idea of group action is that we can think of each element in \(G\) as doing permutations on elements in \(X\). Different group element might perform different pemutation on the set \(X\) but we have the additional condition that if you let \(g\) permute elements in \(X\) and you let \(h\) permute the result of \(g\), then their combine permutation is the same as how the permutation that \(hg\) would have done on \(X\).
Let's get on with our example of necklace. The idea is to define a group action on some set about the constructed necklaces. So we first define that set. As we have \(11\) gems, we can think of the necklace as having \(11\) positions labeled from \(1\) to \(11\) where we can place gems in it. Each of the different ways of putting the gems on the necklaces will be an element in our set \(X\). And Voila! we have constructed our set \(X\) for which a group will act on! Now, let's count the number of elements in \(X\).
There are:
  • \(2\) placeholders out of \(11\) for the \(2\) blue gems: \(\binom{11}{2}\)
  • \(2\) placeholders for the remaining \(9\) for the green gems: \(\binom{9}{2}\)
  • \(7\) placeholders out of the remaining \(7\):\(\binom{7}{7}=1 \)
for a total of \( \binom{11}{2} \cdot \binom{9}{2} \cdot \binom{7}{7} = 1980 \) elements in \(X\). Now we need to introduce a group to act on \(X\). We can consider the group \(Z_{11}\) acting on \(X\). It acts as follow: element \(i\) shift all gems position on the necklace by \(i\) clockwise. Namely, suppose we had the necklace in the necklace config 1, then the element \(1 \in Z_{11} \) modify it to:

Do you see what is happening? the element \(i \in \mathbb{Z}_{11} , i < 11 \) just shift all gems on the necklace by \(i\) position. And this is how \(G\) act on \(X\). To see that this is indeed an action, we need to verify that the two conditions that has to be satisfied in the definition of group action is satisfied.
  1. The identity element \(0 \in \mathbb{Z}_{11}\) fixes all configuration in \(X\) because it dosen't shift the positions of the gems
  2. Given \(i,j \in \mathbb{Z}_{11}\), shifting the gems by \(i\) position clockwise on the necklace and then shift the gems by \(j\) position clockwise has the same effect of shifting them by \(i+j\) position.
    3. And so this shifting operation is indeed a group action of \(\mathbb{Z}_{11}\) to \(X\).
    We said in the beginning that we can think of an element in the group as doing permutation on the elements in the set \(X\). So here is the formal lemma and proof.
    (Proposition 2.4) Given a group action \(\phi\) of group \(G\) on set \(X\), the function \(f_{\phi} : x \to \phi(g,x) \) is permutation!
    Well, \(gx=gy \iff g^{-1}g(x) =g^{-1}g(y) \iff ex=ey \iff x=y\) and given \(x \in X\), \(g(g^{-1}x)=x\).
    Given an action \(\phi\) of \(G\) on \(X\), the orbit of an element \(x \in X\) is just \[ \mathcal{O}_x = \{gx|g \in G\}\]
    We think of the orbit of an element \(x \in X\) exactly as what it is, the set of elements in \(X\) that \(x\) get sent to under some permutation. For example, the following necklace is not in the orbit of the previous necklaces under the action of \(\mathbb{Z}_{11}\):

    This is because you cannot shift the gems from the previous configuration to a configuration in which the red gems dosen't form a continous segment of the necklace.
    (proposition 2.9)Given action \(\phi\) of \(G \curvearrowright X\), the orbits of the action partition the set \(X\)
    if \( w \in \mathcal{O}_x \cap \mathcal{O}_y \) then there exist \(g, g' \in G\) such that \(gw=x, g'w=y \Rightarrow g'g^{-1}x=y \Rightarrow \mathcal{O}_y \subset \mathcal{O}_x\). Bu similar argement, \(\mathcal{O}_x \subset \mathcal{O}_x\). Hence if two orbits intersect, they are equal. Also, every element belong to an orbit containing themself so orbits partition the set \(X\).
    (definition 2.6) Given a group action \(\phi\) of group \(G\) to a set \(X\), the \(\textbf{stabilizer}\) of a set element \(x \in X\) is just the set of element in the group that fixes it: \[ S_{xx} = \{g \in G| gx=x\}\]
    For example, the stabilizer of necklace configuration \(1\) is just the identity element \(0\).
    The stabilizer of \(x \in X\) for an action \(\phi\) of \(G\) on \(X\) is a subgroup of \(G\).
    \( g,h \in stab(x) \Rightarrow ghx=gx=x \Rightarrow gh \in stab(x), e(x)=x \Rightarrow e \in stab(x)\) So \(stab{x}\) is closed under group operation and contains \(0\) so it is a subgroup.
    (difinition 2.7) Given a group action \(\phi\) of a group \(G\) on a set \(X\), the \(\textbf{transform}\) of two elements \(x,y \in X\) is defined as: \[S_{xy} = \{g \in G| gx=y\}\]
    For example, \(2\) lies in the transformer of the necklace config1 and necklace config 3 in the above example.
    (2.8) Given a group action \(\phi\) of \(G\) on a set \(X\), the quotient of \(\phi\) is just the set of all orbits of the action : \[X/G = \{ \mathcal{O}_x |x \in X\} \]
    (theorem 3.1, Orbit-Stabilizer theorem): Given group action \(\phi\) of \(G\) on \(X\) \(G \curvearrowright X\), then the number of element in orbit of \(x \in X\) is the number of element in \(G\) over its stabilizer \(S_{xx}\) \[|G| = |S_{xx}||\mathcal{O}_x|\]
    We can associate to each element \(y \in \mathcal{O}_x\) a coset of \(S_{xx}\). If \(gx=y\) then elements in coset \(gS_{xx}\) send \(x\) to \(y\). So this is the correspondence beween cosets of \(S_{xx}\) and elements \(\mathcal{O}_{x}\). We know number of cosets of \(S_{xx}\) is \(\frac{|G|}{|S_{xx}|}\) and so that is the number of elements in the orbit \(\mathcal{O}_x\).
    (theorem 3.2, Burnside's Lemma) : Given a finite group \(G\) and a finite set \(X\) and an action \(\phi\) of \(G\) on \(X\), then the number of orbits of this action is : \[|X/G| = \frac{1}{|G|}\sum_{g \in G}|X^g| \] Here, \(X^g=\{x \in X|gx=x\}\) is the set of elements fixed by action of \(g\).
    \(\sum_{g \in G}|X^g| \) is the sum of number of elements in \(X\) fixed by each element \( g\in G\) and so we can count them in a different way, namely, it is the sum of number of elements in the stabilizer of each \(x \in X\), i.e, \(\sum_{x \in X} |S_{xx}| \). Now we do: \[ \frac{1}{|G|} \sum_{x \in X} |S_{xx}| = \frac{1}{|G|} \sum_{x \in X} \frac{|G|}{|\mathcal{O}_x|}=\sum_{x \in X}\frac{1}{|\mathcal{O}_x|}=|X/G|\]
    So now we are ready to solve our problem of counting number of necklaces. So consider the action \(\phi\) of the group \(\mathbb{Z}_{11}\) on the set of constructed necklaces \(X\) as defined above. we have : \[\begin{cases} |X^0|=1980\\ |X^i|=0 \forall i \in [11]-\{0\} \end{cases} \] ok so why is it that \( |X^i| \) don't fix anything? It is because if \(i\) fix \(x \in X\), then all elements in \(\langle x \rangle =\{x^n |n \in \mathbb{N} \}\) fix \(x\). \(\langle x \rangle \) is a subgroup of \(\mathcal{Z}_{11}\) but size of subgroup is a divisor of size of th group(Lagrange theorem). So \( \langle x \rangle = \mathbb{Z}_{11}\). this means that \(1\) fixes \(x\). But any necklace fixed by shifting the gems by \(1\) position must be monochromatic. Hence a contradiction and \(i\) don't fix anything. So by Burnside lemma, we have \(|X/G|=\frac{1980}{11}=180\) different necklaces.

    Let \(p\) be a permutation on \(X\). the \(\textbf{type}\) of \(p\) is the set \(\{b_1,b_2,..,b_n\}\) where \(b_i\) is the number of cycles of length \(i\) in the cycle decomposiiton of \(p\).

    Given group action \(\phi\) of \(G\) to \(X\) where \(|X|=n\), \(\textbf{the Cycle index polynomial}\) is defined as: \[Z_{\phi}(x_1,...,x_n) = \frac{1}{|G|}\sum_{g \in G}\prod_{i=1}^n x_i^{b_i(g)}\] Here, \(b_i(g)\) is the \(i^{th}\) element of the type of the permutation induced by \(g\).

We group the CGF by weight to get: \[ \begin{gather*} CGF & = \sum_{c \in C}W(c) \\ & = \sum_w w|A(w)| \\ & = \sum_w \frac{w}{|G|} \sum_{g \in G} |S_{gg}(w)| \\ & = \frac{1}{|G|}\sum_{g \in G} \sum_w w|S_{gg}(w)| \\ & = \frac{1}{|G|} \sum_{g \in G} \sum_{f \in S_{gg}} W(f) \end{gather*} \] In our example, all orbits have different weight, so our example cannot fully express the use of the notation \(A(w)\) that is more useful when there are multiple orbits with the same weight.
    Given \( f \in S_{gg} \), \(f\) must be constant on each of the cycles induced by \(g\) on \(X\), and so : \[\sum_{f \in S_{gg}} W(f) = \sum_{f \in S_{gg}} \prod_{i=1}^{k(g)} w(f(x_i))^{|C_i|}\] Here, \(C_1,...,C_k\) are the cycle decomposition of\(f\) and \(x_i\) is some element in \(C_i\).
    Before we move on, we look at the expansion of formula like :\((a_1+a_2+...+a_m)^n\), this formulat is just the sum of terms each having \(n a_i\)'s like for example: \[ (a_1+a_2+a_3)^3= a_1a_1a_1+a_1a_1a_2+a_1a_1a_3+a_1a_2a_1+...a_3a_3a_3\] The case when we are taking the \(n^{th}\) power of sum of two variable is called binomial theorem.
