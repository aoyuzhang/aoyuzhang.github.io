Given two polynomials \(f(x),g(x)\) having real roots \(r_1 \leq r_2 \leq... \leq r_n \) and \(s_1 \leq s_2 \leq ... \leq s_{n-1} \) we say \(f\) and \(g\) interlace if and only if: \[r_1 \leq s_1 \leq r_2 \leq ... \leq s_{n-1} \leq t_n \]
The roots of polynimials \(f,g\) interlace if and only if the linear combinations \(f + \alpha g\) have all real roots for all \( \alpha \in \mathbb{R}\).
Given Hermitian matrix \(A\), the \(\textbf{principal submatrix}\) of \(A\) is a square submatrix of \(A\) obtained by deleting some rows and some columns of \(A\) so that the first \(k\) rows and columns, for some number \(k\), are the ones that remain.
